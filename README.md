<h2 align='center'><samp>Sharptoy</samp></h2>

<p align='center'>
C# playground for drawing things to a canvas. Runs locally from your web browser, no server necessary. Diagnostics and completions are supported.</p>

## Live demo

https://sharptoy.net/

> [!NOTE]
> No effort has gone into design or theming.
> This is a recently forked version of Jake Yallop's WasmSharp playground that adds an `Input`, `Context2D`, and `Screen` API for making interactive C# programs on the web.
